5 HTML tags I've learnt

Paragraph

<p> "p" stands for a paragraph. It's a paired tag. You can put texts here, then it will show up.</p>

Image

Tag format: <img src="absolute-path-to-the-image" alt="alternate text"/>

"img" stands for image. You should always remember to add "src" and "alt" those two attributes; One for relative or absolute path to the image, and other provide description for search engines and screen readers in case the image doesn't load.

Anchor

Tag format: <a href="link">Anchor name</a>

"a" stands for anchor. Remember href is an attribute of the tag.

Table container

<table></table> is the table container. If you want to add a table to the page you should start with this tage.
